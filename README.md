# Portfolio di Stefano Carbone

Portfolio statico creato a partire da CV, tesi e documentazione dei progetti presenti nella cartella.

## Avvio locale

Apri `index.html` direttamente nel browser, oppure avvia un server statico:

```powershell
python -m http.server 8000
```

Poi visita `http://localhost:8000`.

## File principali

- `index.html`: struttura della pagina
- `styles.css`: stile e responsive layout
- `script.js`: animazioni di reveal in scroll
- `assets/docs/`: CV e PDF dei progetti collegati dal sito

## Personalizzazioni utili

- Aggiungere GitHub e LinkedIn nella sezione contatti
- Sostituire o integrare i link dei progetti con repository pubblici o demo live
- Inserire screenshot reali dei progetti per rafforzare la sezione portfolio
